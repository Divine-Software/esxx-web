ESXX is available in several different packages that can be downloaded manually. We also host a couple of package repositories for Linux and OpenSolaris.

For OSX, the recommended way to stay updated is to subscribe to ESXX on i use this with AppFresh.

Direct download

If you just wish to check it out, a manual download from BerliOS is probably the best option.

CentOS, Fedora and RHEL, using yum

Download esxx.repo (or copy and paste the text below) and save it as /etc/yum.repos.d/esxx.repo.

[esxx]
name=ESXX
baseurl=http://esxx.org/repos/rpm/
enabled=1
gpgcheck=0

Debian and Ubuntu, using apt-get

OpenSolaris, using pkg (IPS)

Add the ESXX IPS repository using the package manager or with the following command line:

pkg set-authority -O http://esxx.org/repos/ips/ esxx.org
